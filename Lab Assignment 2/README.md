# 🎮 Number Guessing Game - Flutter Application

A complete, production-ready Flutter application that implements a Number Guessing Game with SQLite database integration for storing and managing game results.

## ✨ Features

### Core Game Features
- 🎲 **Random Number Generation** - Generates a random number between 1-100
- 🎯 **Guess Input** - User-friendly input field with validation
- 📊 **Real-time Feedback** - Immediate response to each guess
- 🟢 **Color-Coded Responses**: Green for correct, Red for too high, Orange for too low
- 📈 **Attempt Tracking** - Counts total guesses made

### Database Features
- 💾 **SQLite Integration** - Persistent storage of all game results
- 📋 **Game History** - View all previous game attempts
- 🗑️ **Record Management** - Delete individual or all records
- ⏰ **Timestamps** - Record date/time for each guess

### User Interface
- 🎨 **Material Design 3** - Modern, clean interface
- 📱 **Multi-Screen Navigation** - Bottom tab navigation
- 🎪 **Responsive Design** - Works on all screen sizes
- ✅ **Input Validation** - Comprehensive error handling

## 🎮 How to Play

1. Launch app → "Play Game" tab
2. Read instructions
3. Enter a number (1-100)
4. Click "Make Guess" or press Enter
5. Get feedback (Correct/Too High/Too Low)
6. Repeat until you win
7. Check "History" tab to see all attempts
